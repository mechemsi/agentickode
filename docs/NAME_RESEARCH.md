## Domain Registration Checklist

When a name is chosen, register immediately:

- [ ] .com domain
- [ ] .dev domain
- [ ] .io domain (optional)
- [ ] .ai domain (optional)
- [ ] npm package name (`npm init` + `npm publish --access public`)
- [ ] PyPI package name
- [ ] GitHub organization
- [ ] Docker Hub namespace
- [ ] X/Twitter handle
