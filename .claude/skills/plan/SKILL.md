# Plan Skill
When the user runs /plan:
1. Ask for the feature name if not provided
2. Immediately create `docs/plans/<feature-name>.md`
3. Write the plan directly to the file with phases, tasks, and file lists
4. Do NOT enter plan mode or ask for approval
5. Keep exploration to under 2 minutes
6. Output: "Plan written to docs/plans/<feature-name>.md"
